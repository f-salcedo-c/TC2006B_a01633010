Tags de Contenido

Estos tags permiten ingresar contenido al documento.


El tag <img> permite añadir imagenes desde una fuente. Es un tag sin cierre pues no envuelve contenido. Por ejemplo;

<img src="https://picsum.photos/200/200">


El tag <br> hace un line break en el documento. Por ejemplo;

<br>


El tag <p> permite ingresar texto sin formato en un párrafo. Por ejemplo;

<p>Hello, World!</p>

Hello, World!


El tag <a> permite ingresar hipervinculos sin formato. Por ejemplo;

<a href="https://www.google.com" >Click here!</a>

Click here!

Tags de Estructura

Estos tags no demuestran contenido en el documento pero sirven para delimitar el documento como tal. Ningún tag tiene una funcionalidad especial, pero su nombre ayuda a localizar el tipo de contenido que enmarca.


El tag <div> señala una divisíon en el documento.

El tag <span> señala una enmarcación de una sección de texto.

El tag <header> señala un contenedor de introducción a alguna sección.

El tag <section> señala un contenedor de una sección.

El tag <article> señala un contenedor con contenido independiente al documento, pudiera ser trasladado a otro sin cambiar su estilo o formato.

El tag <aside> señala un contenedor con contenido aparte del otro donde se encuentra.

El tag <footer> señala un contenedor de pie a alguna sección.

El tag <nav> señala un contenedor con ligas o elementos de navegación.

Estos tags permiten la definición y construcción de tablas.


El tag <table> señala el inicio y fin de una tabla HTML - que contiene un tag table y mínimo uno de los siguientes.

El tag <tr> señala una fila en la tabla, que puede ser de distintos tipos como,

El tag <th> señala el encabezado de una tabla.

Tags de Listas

Estos tags permiten la definición y construcción de listas HTML. Una lista puede ser de distintos tipos como,


El tag <ul> define el inicio y fin de una lista HTML desordenada.

El tag <ol> define el inicio y fin de una lista HTML ordenada.

Estas listas se conforman por mínimo uno de los siguientes,

El tag <li> define un elemento de la lista.

Por ejemplo,

<ol>
<li> First Element </li>
<li> Second Element </li>
<li> Third Element </li>
</ol>

<ul>
<li> Some Element </li>
<li> Another Element </li>
<li> This Other Element </li>
</ul>

  1. First Element
  2. Second Element
  3. Third Element
  • Some Element
  • Another Element
  • This Other Element
