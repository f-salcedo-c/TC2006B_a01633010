Ligando una hoja de estilo al documento

Para ligar una hoja de estilo se puede definir el estilo utilizando el tag <style> dentro del tag <head>.

Para ligar una hoja de estilo externa se puede definir el recurso utilizando el tag <link> dentro del tagh <head> y definiendo la ubicación en su atributo href.

Border

La regla border permite la definición de enmarcado de un elemento.

Tambien las reglas border-left border-right border-top border-bottom permite definir el enmarcado en cada lado.

La regla border-width permite la definición de tamaño de enmarcado

La regla border-radius permite la definición de enmarcados redondeados a cierto radio.

La regla border-color permite la definición de color de enmarcado.

La regla border-style premite la definición del estilo de linea de enmarcado.

El siguiente elemento div tiene las siguientes reglas

div {
border-width: 10px; border-color: red;
border-style: solid;
border-radius: 10px;
}

Que es lo mismo que,

div {
border: 10px solid red;
border-radius: 10px;
}

Estilo de Texto

La regla font permite la definición de estilo de texto.

La regla font-size permite la definición del tamaño de fuente en diversas medidas.

La regla font-family permite la definición del tipo de fuente a utilizar.

La regla font-style permite la definición del estilo de fuente, si existe dentro de la familia escogida.

La regla font-weight permite la definición de peso de la fuente.

La regla color permite la definición de color en RGB o HEX o Shorthand del texto.

El siguiente elemento p tiene las siguientes reglas

p {
font-size: 40pt;
font-family: Helvetica;
font-style: italic;
font-weight: 400;
color: pink;
}

Que es lo mismo que,

p {
font: italic 400 40pt Helvetica;
color: pink;
}

I am a container!


Estilo de Lista

La regla list-style permite la definición de estilo de lista.

La regla list-style-type permite la definición de apariencia del listado.

La regla list-style-image permite la definición de apariencia del listado con una imagen.

La regla list-style-position permite la definición de colocación del listado partiendo del encabezado.

El siguiente elemento ul tiene las siguientes reglas

ul {
list-style-type: square;
list-style-position: outside;
}

Que es lo mismo que,

ul {
list-style: square outside;
}

Estilo de Contenido

La regla background-image permite la definición de una imagen como fondo del elemento.

La regla width permite la definición de tamaño del ancho del elemento con diversas medidas.

La regla height permite la definición de tamaño del alto del elemento con diversas medidas.

La regla overflow permite la definición de como mostrar contenido excedente del tamaño de la ventana.

La regla filter permite la definición de filtros gráficos sobre el contenido.

El siguiente elemento div tiene las siguientes reglas

div {
background-image: url(https://picsum.photos/200) ;
width: 200%;
height: 50px;
overflow: hidden;
filter: blur(10px);
}

Estilo de Posición

La regla margin permite la definición de tamaño del margen (espacio desde el fin del borde del elemento).

La regla padding permite la definición de tamaño del relleno (espacio desde el fin del elemento hasta el borde del elemento).

La regla display permite la definición de como mostrar el elemento en la ventana.

La regla position permite la definición de la posición del elemento dentro de su contenedor.

Los siguientes elementos img tiene las siguientes reglas

img {
margin: 20px;
border: 10px solid black; padding: 10px;
display: inline; position: static; }
